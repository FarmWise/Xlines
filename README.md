# X-Lines

*Cluster 2D points in parallel lines*

## Requirements

Python 3.x

## Installation

Using pip
```
pip install git+https://github.com/sds-dubois/Xlines.git
```

Or clone and install with
```
python setup.py install
```


## Getting started 

Check the [example](/examples/Example.ipynb).
